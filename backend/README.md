# Backend

This is the NestJS backend for Wellnix-EHR-EMR-Track0342.

## Tech Stack

- NestJS
- TypeScript
- TypeORM
- PostgreSQL

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```
2. Configure PostgreSQL connection in `.env` or `ormconfig.json`.
3. Run the development server:
   ```bash
   npm run start:dev
   ```
